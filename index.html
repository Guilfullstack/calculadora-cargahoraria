Calculadora de carga horária

Como usar o sistema

O objetivo do meu site é oferecer uma forma rápida e fácil para as ligas acadêmicas calcularem a carga horária das suas atividades e dos seus membros na hora de entregar relatórios.

Para que o sistema funcione corretamente:

  • Os arquivos devem estar no formato CSV.
  • Devem conter uma coluna com o nome Carga Horária com o valor numérico da carga horária da atividade.
  • Uma coluna Membros que deve conter o nome dos membros. Caso a atividade tenha mais de um membro, os nomes devem estar entre aspas e separados por vírgulas.

Caso você use um dos nossos modelos de gerenciamento do Notion, não será preciso alterar nada. Apenas exporte no formato correto e carregue os arquivos aqui: Baixe os modelos aqui.
